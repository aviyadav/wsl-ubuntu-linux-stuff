# Fun things

#### fastfetch

[Ubuntu]

```
sudo add-apt-repository ppa:zhangsongcui3371/fastfetch
sudo apt update
sudo apt install fastfetch
```

[OpenSUSE-Tumbleweed]

```
sudo zypper install fastfetch
```

[fedoraLinus-43]

```
sudo dnf install fastfetch
```
